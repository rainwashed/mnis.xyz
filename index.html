My Projects

wtt

wtt (Windows Terminal Terminal) is an application I wrote in JavaScript, originally Python, which is used to simplify the process of adding themes and backing up the Windows Terminal configuration file. It features an easy-to-use CLI and is easily modifiable in the helpers/utils.js.

chalk-flags

Chalk Flags is an API that allows for the conversion of a specially-formatted string into terminal ASCII styles with the style-declaration being in the text itself. The project was rewritten during my winter break due to me having the time and motivation to redo the framework, but for now, it is not being worked on.
